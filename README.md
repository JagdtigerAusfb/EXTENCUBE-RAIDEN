# EXTENCUBE-RAIDEN

**[WORK IN PROGRESS]**

This repository contains everything related to the Rubik's Cube solving robot, Raiden. This project was developed as an extension project at the State University of Campinas (UNICAMP).

## Introduction

The goal of this project was to develop a simple Rubik's Cube solving robot using accessible, easy-to-use, and relatively inexpensive components.
